# Spartan-Gold-Python
A Python port of Professor Thomas Austin's Spartan-Gold JavaScript blockchain based-currency.

## Setup
1. Clone the repo
git clone https://github.com/gabeP232/Spartan-Gold-Python.git

2. Create and activate virtual environment
python -m venv venv
venv\Scripts\activate

3. Install dependencies
pip install -r python-port/requirements.txt
